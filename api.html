kit.Flask(module_name, path=None)[source]¶

Returns the Flask application registered for the module.

Parameters:
  • module_name (str) – the module name for which to get the application. Typically, this should always be set to __name__.
  • path (str) – the path to the overall kit configuration file. This can be used to load the Flask application outside of the kit command line tool.
Return type:

flask.Flask

Note

This isn’t a class constructor.
